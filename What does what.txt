NewWorker
- Bereichnet einen normalen Run oder TaggEff Run
- Parameter:
            FileID
            -b ProgramBatchMode
            -c ProgramOnlyFileCheckMode
            -s ProgramSaveSpectraMode
            -n ProgramNoTBrowserMode und ProgramBatchMode
- schreibt die histogrammierten Ergebnisse als root Datei nach /datapool in die jeweiligen
  Verzeichnisse unter output

AR
- Ruft den NewWorker/build/bin/NewWorker auf. 2 Parameter: Start und End FileID

CBEnergyFitter
- Berechnet die Position und Breite der Mesonen-Masse für eine Beamtime, zuerst muss NewWorker diese Dateien erzeugt haben.
- Parameter: 
            SelectedBeamtime (0, 1, 2)
            -b    ProgramBatchMode
            -d    ProgramDebugMode
            -new  Wenn die Spalte CBEnergyScale in RunsMetaInformation.dat neu mit dem Wert 1.0 gefüllt werden soll
            -r    ReplaceRunMetaInfoFile
- Programm 1x ausführen:
  CBEnergyFitter/build/bin/CBEnergyFitter -r -b




NewTaggEff
- Berechnet die TaggEff für eine Beamtime, zuerst muss NewWorker diese Dateien erzeugt haben.
- Parameter: 
            SelectedBeamtime (0, 1, 2)
            -b    ProgramBatchMode
            -d    ProgramDebugMode
            -new  WriteAbsTaggEffNew -> Wenn die Spalte BeamPolDegree in RunsMetaInformation.dat neu mit dem Wert 1/LaddP2 Ratio gefüllt werden soll
            -r    ReplaceRunMetaInfoFile
- Programm 1x für jede Strahlzeit ausführen:
  NewTaggEff/build/bin/NewWorker -r -b 0
  NewTaggEff/build/bin/NewWorker -r -b 1
  NewTaggEff/build/bin/NewWorker -r -b 2
